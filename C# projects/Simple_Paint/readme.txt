Упрощённый аналог Paint.

Запускается ехе-шником по пути Simple_Paint/Simple_Paint/bin/Debug/Simple_Paint.exe

Сначала рисует "кистью", можно выбрать несколько фигур, заливку, сохранять и открывать графические файлы.

Алгоритм заливка выполненн рекурсивным алгоритмом заливки на основе серий пикселов (линий). 

При нажатии на область pictureBoxMain выполняется функция, в которой выполняется уссловия через switch case. В зависимости от выбранного элемента вызывается определённая функция, которая рисует выбранный элемент.
